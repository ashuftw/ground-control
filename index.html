How to get your Client ID:
  1. Open the Google Cloud Console.
  2. Create a project and enable the Google Calendar API and Google Drive API.
  3. Go to "OAuth Consent Screen", choose External, add your email under Test Users, and add scopes for drive.appdata and calendar.events.
  4. Go to "Credentials", click "Create Credentials" → "OAuth client ID" (select "Web Application").
  5. Add http://localhost:8000 (or your current web URL) to the Authorized JavaScript origins.
  6. Copy the Client ID and paste it here.
